## Quick Start

```bash
git clone https://github.com/KaitoJD/About.git
cd About
npm install
npm run dev
```

## Commands

```bash
npm run dev          # Development server
npm run build        # Production build
npm run build:github # GitHub Pages build
npm run lint         # Linting
```
